## 📚 核心模块（按考场分值 / 难度排的）

#### 🎯 基础必拿分 （占 40% 左右，必须全对）

**二分查找** — 精确查找、左右边界、二分答案、旋转数组、峰值查找、高精度方程求根。填空和编程都爱考，模板背熟直接秒。

**BFS 广搜** — 迷宫最短路径、多源 BFS、双向 BFS、路径回溯。这类题就是套模板，写对就能拿分。

**DFS 回溯** — 全排列、组合、子集、N 皇后、回文分割、幻方构造。暴力枚举的万能方法，适合骗分也适合满分。

⭐ 考场必对，先拿稳

#### 🚀 进阶核心分 （占 45% 左右，决定拉不拉分）

**一维 DP** — 线性递推、最大子段和、最长上升子序列、01 / 完全背包、背包计数。动态规划入门必须搞透。

**二维 DP** — 数字三角形、最长公共子序列、编辑距离、二维费用背包、方格取数、三维 DP。省赛压轴和国赛常客，这块拿住就稳了。

💡 拉分关键，多花时间

#### ⚡ 临考提速神器 （占 15%，省时间用的）

Python 内置函数速查、日期处理、质因数分解、差分数组、异或和、滑动窗口、单调栈。考场上直接拿来用，不用自己从零手写，省下半小时检查。

⏱️ 考前背熟，考场复用
